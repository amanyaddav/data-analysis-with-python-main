# Data Analysis with Python

This repository contains beginner to intermediate data analysis projects using Python.  
The projects demonstrate data cleaning, data visualization, and exploratory data analysis (EDA).

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Projects Included

### 1. Sales Data Analysis
- Cleaned raw sales dataset
- Performed exploratory data analysis
- Visualized monthly sales trends

### 2. Student Performance Analysis
- Analyzed student score dataset
- Compared performance based on gender and study hours
- Generated insights using graphs

### 3. COVID-19 Data Analysis
- Processed global COVID dataset
- Visualized cases and deaths
- Created trend graphs

- Exploratory Data Analysis
- Python Programming
